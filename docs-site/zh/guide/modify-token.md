# 修改令牌设置

令牌创建后，你可以根据用途调整额度、模型范围、分组和过期时间。合理拆分令牌能让用量更清楚，也更容易在出现异常时快速停用。

## 常见设置

| 设置 | 作用 |
| --- | --- |
| 名称 | 用来区分不同客户端或项目 |
| 额度 | 控制该 Key 最多能消耗多少 |
| 过期时间 | 适合临时测试、短期共享或活动使用 |
| 模型限制 | 限制只能调用指定模型 |
| 分组 | 决定价格策略、可用渠道和工具兼容性 |
| IP 限制 | 只允许固定出口 IP 调用 |

## 推荐做法

- 每个工具单独建一个 Key，例如 `claude-code` 和 `codex` 分开。
- 临时测试 Key 设置较短过期时间和较低额度。
- 生产服务 Key 设置 IP 限制。
- 如果怀疑 Key 泄露，立即禁用或删除。

## 修改后不生效

客户端可能缓存了旧配置。修改令牌后，建议重启对应工具或重新加载配置。
