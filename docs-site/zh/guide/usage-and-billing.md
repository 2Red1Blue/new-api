# 费用与用量

laiber.cloud 会根据你调用的模型、输入输出 token、令牌分组和缓存命中情况计算费用。你可以在控制台查看请求日志和额度变化。

## 你可以看到什么

- 当前余额或额度。
- 每个 API Key 的消耗。
- 每次请求的模型、token、费用、耗时和错误。
- 按模型和令牌维度统计的用量。

## 排查费用异常

如果某次调用费用看起来不对，建议检查：

- 请求使用的模型名称。
- API Key 所属分组。
- 输入 token 和输出 token。
- 是否命中缓存。
- 日志中的计费明细。
- 是否使用了更高能力或更长上下文的模型。

如果费用仍然无法解释，请带上请求时间和模型名称联系站点支持。
