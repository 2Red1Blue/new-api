# 注册账号

访问 laiber.cloud 控制台即可注册或登录：

```text
https://ai.laiber.cloud
```

注册完成后，你可以创建自己的 API Key，并把它配置到 Claude Code、Codex、Cherry Studio 或其他 OpenAI 兼容客户端中。

## 注册步骤

1. 打开 `https://ai.laiber.cloud`。
2. 点击登录或注册入口。
3. 按页面提示选择可用的注册方式。
4. 完成登录后进入控制台。
5. 前往 `令牌` / `API Keys` 页面创建专属 Key。

::: tip 提示
如果平台开启了邮箱验证，请使用你能长期访问的邮箱。后续找回密码、接收通知和处理账号问题都会更方便。
:::

## 注册后可以做什么

- 创建专属 API Key。
- 查看当前账号可用模型。
- 接入 Claude Code、Codex 或其他客户端。
- 在控制台查看余额、用量和请求日志。

## 常见问题

### 登录后看不到创建 Key 的入口

请先确认账号是否完成注册流程，或者刷新页面后重新进入控制台。如果仍然没有入口，联系站点维护人员处理账号权限。

### 忘记密码怎么办

如果登录页提供找回密码入口，按页面提示通过邮箱重置。若当前站点使用第三方登录，请回到对应第三方账号完成登录。
