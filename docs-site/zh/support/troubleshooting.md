# 排障指南

## 请求返回 401

- 确认 API Key 没有复制错。
- 确认请求头是 `Authorization: Bearer <API Key>`。
- 检查 Key 是否被禁用、过期或额度耗尽。
- 检查 Key 是否限制了模型、分组或 IP。

## 模型不存在

- 调用 `/v1/models` 查看当前 Key 可用模型。
- 检查令牌是否限制了模型范围。
- 确认客户端里的模型名称没有拼写错误。
- 如果模型列表里也没有该模型，请更换一个当前可用模型。

## 请求超时

请求超时通常和网络、模型响应速度或上游繁忙有关。可以按下面顺序处理：

- 先重试一次简单问题，排除临时波动。
- 换一个模型测试。
- 检查本地网络和代理。
- 在控制台查看请求日志里的错误信息。
- 如果多个模型都持续超时，联系站点支持。

## 登录状态异常

- 刷新页面后重新登录。
- 清理浏览器缓存或换一个浏览器测试。
- 确认账号没有使用错误的第三方登录方式。
- 如果仍然异常，联系站点支持。
